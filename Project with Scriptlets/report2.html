ΑΝΑΦΟΡΑ ΕΡΓΑΣΙΑΣ PROJECT Shop

 

Στην παρούσα εργασία καλούμαστε να δημιουργήσουμε δύο JSP σελίδες η οποίες θα συμπεριφέρονται σαν ένα Shop στο οποίο αγοράζουμε κάποια προϊόντα και μας εμφανίζουν την τελική τους τιμή, όπως μια συνήθης εφαρμογή shopping cart σε κάποιο site του διαδικτύου. Οι απαιτήσεις είναι οι εξής: Στην πρώτη σελίδα θα χρησιμοποιήσουμε JSP με τη χρήση scriptlets, ενώ στη δεύτερη η χρήση scriptlets απαγορεύται και αντί αυτών θα κάνουμε χρήση της Expression Language σε συνδυασμό με java beans. Και στις δύο περιπτώσεις δεν απαιτείται η χρήση servlets για τον υπολογισμό των τιμών, αλλά απαιτείται η χρήση cookies ή χρήση scope, τέτοιων ώστε να διατηρούνται οι τιμές που έχει εισάγει ο χρήστης.

Στην πρώτη περίπτωση, αφού διαμορφώσαμε οπτικά τη σελίδα μας, εισάγουμε τις τιμές των προιόντων απευθείας, χωρίς την παρεμβολή του χρήστη, έχοντάς τις δηλώσει ως context parameters στο web.xml αρχείο μας. Στη συνέχεια ο χρήστης εισάγει την ποσότητα του κάθε προϊόντος και πατώντας το κουμπί add to cart εμφανίζονται στην τελευταία στήλη οι τελικές τους τιμές, καθώς και το τελικό άθροισμά τους. Όλη η δουλειά γίνεται με τη χρήση scripts ενσωματομένων στη jsp σελίδα, χωρίς τη χρήση κάποιου servlet. Απαίτηση του project ήταν επίσης σε κάθε session να διατηρούνται οι τιμές που έχει εισάγει ο χρήστης ακόμα και αν κλείσει ο browser και ανοίξει ξανά. Αυτό επιτυγχάνεται με τη χρήση cookies ο κώδικας των οποίων εισάγεται και πάλι με τη χρήση scripts στη jsp σελίδα μας. Τέλος δημιουργήσαμε και δύο κουμπιά τα οποία εμφανίζουν την αναφορά της εργασίας και τον κώδικά της, κάνοντας χρήση Listener και servlet αντίστοιχα.

Στη δεύτερη περίπτωση ακολουθούμε την ίδια διαδικασία για την εισαγωγή των τιμών των προϊόντων. Για την εισαγωγή όμως των ποσοτήτων αυτή τη φορά δημιουργούμε τα λεγόμενα java beans, που είναι στην ουσία αντικείμενα στα οποία μπορούμε να θέσουμε ή να διαβάσουμε Properties και Attributes. Έτσι εισάγοντας ο χρήστης την ποσότητα των προϊόντων μπορούμε να αποθηκεύσουμε αυτήν την πληροφορία και να την αξιοποίησουμε αργότερα. Στο πεδία υπολογισμού των τελικών τιμών εμφανίζουμε τα αποτελέσματα εύκολα, αφού η Expression Language μας δίνει τη δυνατότητα να πραγματοποιήσουμε πράξεις μεταξύ των εισαγόμενων τιμών, αφού δημιουργεί από μόνη της τον απαραίτητο κώδικα java για αυτή την δουλειά. Επίσης δημιουργούμε μια κλάση με Cookies προκειμένου, όπως και στην προηγούμενη περίπτωση, να διατηρούνται οι τιμές που έχει εισάγει ο χρήστης ακόμα και αν κλείσει ο browser και ανοίξει ξανά. Παρά τις προσπάθειές μου για χρήση cookies δεν μπόρεσα να βρω τρόπο να τα χρησιμοποιήσω στο jsp μου, με χρήση μόνο της Expression Language. Τέλος όπως και πριν δημιουργούμε δύο κουμπιά τα οποία εμφανίζουν την αναφορά της εργασίας και τον κώδικά της, κάνοντας χρήση Listener και servlet αντίστοιχα.

Για την ολοκλήρωση της εργασίας απαιτήθηκε να ανεβάσουμε τα sources στο network github και συγκεκριμένα στη σελίδα https://github.com/arlianas/Project_Shop όπου υπάρχουν δύο φάκελοι με τις υλοποιήσεις, με και χωρίς τη χρήση scriptlets, καθώς και στον server inf.uth.gr του Πανεπιστημίου Θεσσαλίας προκειμένου το Project μας να είναι προσβάσιμο από μια static ip.

Κατά την υλοποίηση της εργασίας συνάντησα αρκετές δυσκολίες που αφορούσαν την ανάπτυξη του κώδικα για τα cookies. Επίσης για την υλοποίηση της εργασίας χρησιμοποίησα τη βοήθεια του λογισμικού Netbeans και τις χρήσιμες παρατηρήσεις του συμφοιτητή μου Φώτη Γκέλη, καθώς επίσης και τη βοήθεια που παρέχονταν από τα σχόλια φοιτητών και βοηθών διδασκαλίας στη σελίδα του μαθήματος. Πολλές πληροφορίες για τη λειτουργία των shopping carts βρήκα και σε διάφορα sites και forums στο διαδίκτυο.
